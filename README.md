# Proyecto Clase 2 - Módulo 7

Este repositorio corresponde a la **Clase 4 del Módulo 7** del curso.
El objetivo es aplicar los conceptos vistos en clase y reforzar el aprendizaje práctico.

---

## 🚀 Objetivos del proyecto

- Practicar los fundamentos de **Node.js y Express**.
- Implementar rutas y manejar errores básicos.
- Trabajar con archivos y estructuras de datos en formato **JSON**.
- Consolidar buenas prácticas en la organización del código.

---

## 📂 Estructura del repositorio

```
├── src/              # Código fuente principal
│   ├── routes/       # Definición de rutas
│   ├── controllers/  # Lógica de negocio
│   └── utils/        # Funciones auxiliares
├── data/             # Archivos JSON de prueba
├── package.json      # Dependencias y scripts
└── README.md         # Documentación del proyecto
```

---

## ⚙️ Instalación y uso

1. Clonar el repositorio:
   ```bash
   git clone https://github.com/arlycandia/<nombre-del-proyecto>.git
   ```
2. Instalar dependencias:
   ```bash
   npm install
   ```
3. Ejecutar en modo desarrollo:
   ```bash
   npm run dev
   ```
